Data

Data is a collection of facts, such as numbers, words, measurements, observations or descriptions of things. In computing, data is often stored in databases and can be processed by software applications.

Types of Data

Data Type Function Example
INTEGER Stores whole numbers 42, -7, 1000
FLOAT Stores approximate numeric values with decimals 3.14, -0.001, 2.71828
DECIMAL Stores exact numeric values with fixed decimal points 123.45, -99.99
CHAR Stores fixed-length character strings 'A', 'XYZ'
VARCHAR Stores variable-length character strings 'Hello', 'Database'
TEXT Stores large amounts of text data 'This is a long description.'
DATE Stores calendar dates '2024-06-01'
TIME Stores time of day '14:30:00'
TIMESTAMP Stores date and time information '2024-06-01 14:30:00'
BOOLEAN Stores true or false values TRUE, FALSE
BLOB Stores binary large objects (e.g., images, files) (binary data)
